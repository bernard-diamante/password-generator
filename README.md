# Password Generator App
A mini-project I created to practice basic Kotlin. 
## Features
- Randomized password generation influenced by user-set parameters:
  1. Uppercase letters
  2. Lowercase letters
  3. Numbers
  4. Special characters
  5. Length
- Allows copying to clipboard
